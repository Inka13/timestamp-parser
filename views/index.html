Timestamp Microservice

Example usage:

Append a unix timestamp to the URL of the app like this: https://timestamp-microservice-inka.glitch.me/1450137600

Or a string representing a date like this: https://timestamp-microservice-inka.glitch.me/December 15, 2015

If your input is a valid date, the output will look something like this:

{"unix":1450137600,"natural":"Tue Dec 15 2015"}


Entering invalid date info returns object whose properties values are null:

{"unix":null,"natural":null}
